# kyotoishibumi
Map of 1,470 stone monuments in Kyoto, using Kyoto City open data and egmapjs.

## Demo
[https://code4fukui.github.io/kyotoishibumi/](https://code4fukui.github.io/kyotoishibumi/)

## Features
- Interactive map displaying locations of 1,470 stone monuments in Kyoto
- Click on icons to view monument details, including name, address, and Geo3x3 location code
- Current location can be displayed on the map

## Data / API
- Data source: [石碑リスト | 京都市オープンデータポータルサイト](https://data.city.kyoto.lg.jp/node/14455)
- Data fetched and displayed using the [egmapjs](https://github.com/code4fukui/egmapjs) library
