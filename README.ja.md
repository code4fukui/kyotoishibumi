# 全1470個所 いしぶみマップ

京都市のオープンデータを使用して、京都市内にある石碑の位置を地図上に表示するウェブアプリです。

## デモ
https://code4fukui.github.io/kyotoishibumi/

## 機能
- 京都市内の1470個所の石碑の位置を地図上に表示
- 各石碑の番号、名称、住所、Geo3x3コードを表示
- 現在位置への移動ボタン

## 必要環境
特に必要なものはありません。最新のWebブラウザで動作します。

## 使い方
リポジトリをクローンまたはダウンロードし、index.htmlをWebブラウザで開いてください。

## データ・API
京都市オープンデータポータルサイト「いしぶみリスト」の[データ](https://data.city.kyoto.lg.jp/node/14455)を使用しています。
