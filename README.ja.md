# 全1470個所 いしぶみマップ

京都市のオープンデータを使用して、京都市内にある1,470個所の石碑の位置を地図上に表示するウェブアプリです。

## デモ
https://code4fukui.github.io/kyotoishibumi/

## 機能
- 京都市内の1,470個所の石碑の位置を地図上に表示
- 各石碑の番号、名称、住所、Geo3x3コードを表示
- 現在位置への移動ボタン

## データ・API
- データ出典: [石碑リスト | 京都市オープンデータポータルサイト](https://data.city.kyoto.lg.jp/node/14455)
- [egmapjs](https://github.com/code4fukui/egmapjs) ライブラリを使用してデータを取得・表示
